Github connected

The data set is 16k results of randomly selected Github archive data, selected by events.

Things you should know

  1. This data set is limited to 16k rows.. why? Because Google BigQuery wanted me to pay for more.
  2. The selection was random, and there's no additional meta data attached; only repo name and user name.. why? Because Google wanted me to pay for more.
  3. Built using neo4j and ruby, primarily to find out exactly what neo4j does. Graphs obviously.
